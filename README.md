# Sensor Management System

## Description
This project is a full-stack application designed to manage or monitor sensor data. It features a client-server architecture, demonstrating the integration of backend logic with a web-based frontend interface.

## Tech Stack
* **Backend:** C# (.NET Framework/Core)
* **Frontend:** HTML5, CSS3
* **Project Structure:** Visual Studio Solution (.sln)

## Key Features
* **Client-Server Communication:** Separated logic for the client side and the core sensor processing.
* **Data Visualization:** Clean HTML/CSS layout for potential data presentation.
* **Object-Oriented Design:** Clean C# implementation following SOLID principles.

## How to Run
1. Clone the repository.
2. Open `Sensor.sln` in **Visual Studio**.
3. Restore NuGet packages and build the solution.
4. Run the project.
